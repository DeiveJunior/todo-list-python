Conceito de LOOPS


Loops é um conceito em programação que eu fracassei miseravelmente quando estava aprendendo javascript, mas básicamente é um item se repetir várias vezes, como por exemplo, eu quero repetir o numero 5 varias vezes, então vou criar um laço de repetição até chegar no resultado que eu estou buscando.

While = provavelmente quando se fala em loops, é o conceito mais conhecido. Ele significa "enquanto", bem parecido com as estruturas condicionais.

n = 1

while n <= 10:
    print("*")
    n = n + 1

Acima então básicamente, eu criei um loop. Eu primeiro crio uma váriavel para conter o item que vai ser repetido várias vezes, que no caso vai ser o número 1. Depois eu aplico o conceito do while, dizendo que enquanto a minha variavel for menor ou igual a 10, ela vai se repetir até chegar nesse resultado. Mas para ela se repetir até chegar no resultado, e encerrar o loop, eu preciso daquele elemento que eu utilizei ali "   n = n + 1   ", esse elemento serve para informar ao loop que já está na hora dele parar, se não tiver esse contador, o meu loop vai ser infinito e posso travar o meu navegador ou o meu notebook.
